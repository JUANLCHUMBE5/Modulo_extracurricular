Coloque aqui los banners del modulo de Padres.

Nombres esperados por el sistema:

- BANNER DE HOMBRES.png
- BANNER DE MUJERES.png

El modulo de Padres selecciona el banner segun el campo oficial `sexo`, `genero` o `gender` del estudiante:

- M, Masculino, Hombre, Varon -> BANNER DE HOMBRES.png
- F, Femenino, Mujer -> BANNER DE MUJERES.png

Si no existe dato de sexo/genero o falta la imagen, se muestra el diseno base sin romper la pantalla.
